Climactic Variance by Latitude.

The purpose of this project was to demonstrate how various climactic indicators change depending on latitude. As one approaches the equator the angle of incidence of the sun's rays decreases, meaning that the same amount of sunlight falls on a smaller surface area and delivers more concentrated energy. This variance drives differences in temperature, humidity, cloud cover, and wind speed.

My project used the numpy Python library to generate a series of randomized latitude and longitude coordinate pairs. These coordinates were fed into the Citipy database and Python library to find the nearest city to each coordinate set and discarded duplicate results. I then used the OpenWeatherMap API to retrieve the current weather information for each of the given cities and compiled the resulting data into a Pandas DataFrame.

In the visualization stage of the project I used the MatPlotLib library to create a series of scatter plots, showing how each of the values being tracked (temperature, humidity, cloud cover, and wind speen) varied by latitude. This site shows the compiled data and the resulting visualizations. Analysis of trends and correlations is available on the page for each visualization.
